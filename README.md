# AI-Powered QA Testing Portfolio

This repository contains QA documentation created with AI assistance. It includes test plans, test cases, bug reports, test execution reports, and AI prompts used in the QA process.

## Contents

### 1. Test Plans
Detailed test planning documents for different modules and features.

### 2. Test Cases
Functional, negative, boundary, UI, API, regression, and security test cases.

### 3. Bug Reports
Professional defect reports with steps to reproduce, expected results, actual results, severity, and priority.

### 4. Test Reports
Test execution summaries, defect summaries, risks, findings, and QA sign-off notes.

### 5. AI Prompts
Reusable prompts for QA activities such as test case generation, bug report writing, and test report creation.

## Tools Used

- ChatGPT
- GitHub
- Jira
- Postman
- Selenium
- Excel / Google Sheets

## Purpose

The purpose of this repository is to demonstrate how AI can support a QA engineer in test planning, test design, defect reporting, and test reporting.
